The Notary Project

The Notary Project contains the specifications and tooling for implementing secure supply chain, which drives the Notary specification. The Notary Project is also the umbrella project for the Notary specification and Notation, its reference implementation.


Notary

Notary is a specification for signing and verifying container images and storing them as OCI artifacts using OCI-conformant registries. The latest version is Notary v2 which uses the COSE signature envelope and OCI Reference Types.


Notation

Notation is a reference implementation of the Notary specification and has three components: a Notation Core Go library for the core cryptography of signing and verifying, Notation Go library that builds on the core go library and adds functionality such as pushing and pulling from registries, and a CLI (notation) that uses the go libraries to expose the full implementation.

In most cases, signing, verifying, and moving artifacts happens using notation.
